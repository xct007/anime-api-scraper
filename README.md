# animeku-api-scraper
api scraper animeku
## What working
- **search**
   ```
   get anime by query
   ```
## Not working
- **latest**
  ```
  get latest anime update
  ```
- **detail**
  ```
  get anime detail by id or vid
  ```
## Make test
1. Clone repos
```sh
git clone https://github.com/xct007/animeku-api-scraper
```
2. Install require modules
3. Edit ```src/index.ts``` and run
```sh
ts-node src/index.ts
```
